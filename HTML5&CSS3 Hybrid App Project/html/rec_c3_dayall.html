Course 3 소개

이 코스는 올레10코스와 올레1코스를 경유하는 코스입니다.

중수이상의 코스며, 반드시 페이스조절이 필요한 일정/코스입니다.

1일째에는 라이딩을 위주로 하시면 될것입니다.

2일째에는 올레10코스에 해당되는 곳을 자전거로 돌아보신후

중문보다는 서귀포에서 관광을 하시고 무리하지 않는선에서 더 진행할수 있다면

남원까지 갈 수 있지만 무리는 금물입니다.

3일째에는 오전과 오후를 나누어 오전에는 성산까지만 이동합니다

그리고 오후는 우도를 보겠다면 성산항에서 오후 1~2시 배로 들어가고

막배정도로 나오면 시간이 맞습니다.

만약 일찍 나오셨다면 올레1코스를 자전거로 돌아보셔도 됩니다.

4일째에는 성산~김녕해안도로는 반드시 타시고.

함덕에서 제주까지 2시간여유를 잡고 여유있게 진행하시면 됩니다.
